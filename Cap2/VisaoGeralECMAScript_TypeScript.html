Visão Geral sobre ECMAScript e TypeScript

let e const no lugar de var

com a palavra reservado var podía-se declarar duas variáveis com o mesmo nome. Agora utiliza-se let e essa nova palavra reservada não permite esse erro.

const : palavra reservado com comportamento de constante. Tem valor somente para leitura. No entanto, Ao trabalhar com objetos, um const somente para leitura permite que as propriedades do objeto recebam novos valores ou sejam atualizadas, mas a referência à variável em si (o endereço de referência na memória) não pode ser alterada, o que significa que ela não pode receber um novo valor.

Templates literais

Com eles pode-se concatenar string sem a necessidade de concatenar os valores.

Para usá-los o texto deve estar entre crases `texto`, e para interpolar um valor de variável, basta defini-lo como um sinal de crifrão e chaves ${}
